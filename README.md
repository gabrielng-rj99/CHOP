# Licenses_Manager

 Licenses Manager

## Estrutura dos campos principais

- **Categoria**: Tipo geral do produto (ex: Antivírus, Sistema Operacional, Banco de Dados)
- **Linha**: Marca ou linha principal do produto (ex: Kaspersky, Windows Server, SQL Server)
- **Modelo**: Modelo, edição ou versão específica (ex: Internet Security, 2019 Datacenter, Standard)

Esses campos são usados para organizar e filtrar as licenças cadastradas no sistema.
