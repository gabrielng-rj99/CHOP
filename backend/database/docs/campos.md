# Documentação dos Campos Principais de Licenças

Este documento descreve a finalidade e o uso dos principais campos utilizados para organizar e cadastrar licenças no sistema Licenses Manager.

---

## Campos

### Categoria

- **Descrição:** Representa o tipo geral do produto licenciado.
- **Exemplos:** Antivírus, Sistema Operacional, Banco de Dados, Ferramenta de Backup.
- **Uso:** Permite agrupar licenças por grandes áreas de aplicação ou tecnologia.

---

### Linha

- **Descrição:** Indica a marca, linha ou família principal do produto.
- **Exemplos:** Kaspersky (Antivírus), Windows Server (Sistema Operacional), SQL Server (Banco de Dados).
- **Uso:** Facilita o agrupamento por fornecedor ou linha de produtos, tornando a busca e o filtro mais precisos.

---

### Modelo

- **Descrição:** Especifica o modelo, edição ou versão da licença.
- **Exemplos:** Internet Security (Kaspersky), 2019 Datacenter (Windows Server), Standard (SQL Server).
- **Uso:** Permite detalhar exatamente qual licença está sendo cadastrada, diferenciando versões, edições ou variantes.

---

## Resumo

A combinação dos campos **Categoria**, **Linha** e **Modelo** garante flexibilidade e precisão na organização das licenças, facilitando buscas, relatórios e o gerenciamento eficiente dos ativos de software.
